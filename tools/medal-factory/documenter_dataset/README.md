# Dataset Documenter badge — dataset files

This folder contains a **fully documented** dataset (subtitle, description, keywords, file description, column descriptions) for the [Dataset Documenter](https://www.kaggle.com/progression/badges) badge (usability 10).

## Create the dataset

From the **repo root**:

```bash
python create_documenter_dataset.py
```

Then on Kaggle, open the new dataset and add anything still needed for **usability 10** (see **DATASET_DOCUMENTER_GUIDE.md** in the repo root):

- **Cover image**
- **Source / provenance** (if not in description)
- **Update frequency**

Once the dataset shows **usability 10**, the badge is awarded.
