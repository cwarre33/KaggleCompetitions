# Dataset Pipeline Creator badge

Earn the badge by **creating a dataset from this notebook’s output** on Kaggle.

## Option A: Run this notebook on Kaggle (recommended)

1. **Create a new notebook on Kaggle**  
   Go to [Kaggle Code](https://www.kaggle.com/code) → **New Notebook**.

2. **Copy the code** from the cells in `dataset-pipeline-creator-badge.ipynb` into the new notebook (or upload the .ipynb if your Kaggle supports it).

3. **Run all cells.**  
   They write `pipeline_output.csv` and `README.txt` to `/kaggle/working/`.

4. **Create dataset from output**  
   - In the **right-hand panel**, open the **Output** (or **Data**) tab.  
   - Find **“Create dataset from notebook output”** / **“New dataset from output”** and click it.  
   - Give the dataset a title (e.g. “Pipeline Creator badge”) and create it.

5. The **Dataset Pipeline Creator** badge should appear on your [accomplishments](https://www.kaggle.com/accomplishments) page.

## Option B: Push this notebook via API, then run on Kaggle

From the repo root, you can push this folder as a kernel (after adding `kernel-metadata.json` with your username), then open the notebook on Kaggle and run it, then create the dataset from output as in step 4 above.

## Note

The badge is awarded when Kaggle creates a dataset from a notebook’s **output files**. The notebook must run on Kaggle so that files in `/kaggle/working/` are available to the “Create dataset from output” flow.
