# Model Pipeline Creator badge

Earn the badge by **creating a model from this notebook’s output** on Kaggle.

## Steps

1. **Create a new notebook on Kaggle** at [Kaggle Code](https://www.kaggle.com/code) → New Notebook (or push this folder with `push_model_pipeline_notebook.py` and open it).

2. **Run all cells.** They write `model.pkl` and `README.txt` to `/kaggle/working/`.

3. **Create model from output:**
   - In the **right-hand panel**, open **Output** (or **Data**).
   - Find **“Create model from notebook output”** or **“New model from output”** and click it.
   - Complete the form (title, etc.) and create the model.

4. The **Model Pipeline Creator** badge should appear on your [accomplishments](https://www.kaggle.com/accomplishments).

The notebook must run **on Kaggle** so that files in `/kaggle/working/` are available to the “Create model from output” flow.
